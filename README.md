# Планировщик дня

Fullstack-приложение для планирования задач по дням с временной шкалой, календарём, авторизацией пользователей и приоритетами задач.

## Стек технологий

- Frontend: React + Vite
- Backend: Go
- Database: SQLite
- Containerization: Docker, Docker Compose
- CI: GitHub Actions

## Основные возможности

- регистрация и авторизация пользователей;
- изоляция задач по пользователям;
- создание, редактирование и удаление задач;
- выбор даты и просмотр задач по дням;
- календарь с подсветкой дней, где есть задачи;
- светлая и тёмная тема;
- отображение текущих, следующих и просроченных задач;
- приоритеты задач;
- поддержка задач, переходящих на следующий день;
- Docker-запуск проекта.
